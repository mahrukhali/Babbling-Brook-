Babbling Brook:

For the babbling brook, I used:
  • 3 biquad filters
  • 2 gain nodes
  • 1 brown noise
  • 1 buffer
The biquad filters consisted of: 2 lowpass and one highpass filter.This alone will create the river sound as described above. The lowpass filters frequency were 14 and 400. The high passfilters frequency was 400 as well. When I added the two gain nodes, I was able to hear the babbling brook sound. One gain node which is connected to the highpass filter creates the bubble sound which is set to 800 , this creates the overall sound shaping. The second gain node called gainNode reduces the overall volume to prevent clipping which was 0.1.

Dial Tone:

For the dial tone, I used the frequencies provided in the textbook. I used the following:
  • Two oscillators
  • 1 gain node
To construct the tone,we need to add the sine waves, which occurs by connecting the oscillator to a gain node. I honestly was suprised when I created the tone because it wasnt as complicated as I thought it would be and all I really needed were the right frequencies. The method in the textbook explained what frequencies and waveform I needed, all I needed to do was connect it. The oscillators frequency were 350 and 440. I made the gain node .125 to scale the frequency.

Dial Buttons:

This was the hardest sound for me to create, as the connection to the different options was not working. To create the sounds, I used:
  • Two oscillators
  • 1 gain node
  • 1 biquad filter
Each option has tone pairings of 8 frequencies. I created an object that holds key-value pairs, where each key is a number or symbol, and each value is another object containing the two frequencies. I connected the two pairings to the gain node which connects to the biquad filter. This could have been implemented without the biquad filter, however it just sounds softer with it. The gain node again was set to .125 for scaling.
